## PHP-Perpustakaan
Aplikasi Perpustakaan sederhana berbasis Web

## Fitur
- 3 Tipe User (admin, staff & anggota)
- Peminjaman,Pengembalian
- Add buku, staff
- Filter data buku, katalog buku
- History Peminjaman
- Report
- etc

## Tech Stack
- PHP v7.4
- MySQL (MySQLi)
- JQuery v3.7
- Bootstrap v4
- TCPDF
